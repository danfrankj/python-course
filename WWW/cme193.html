CME 193: Introduction to Scientific Python

Course description: This short course runs for the first three weeks of the quarter and is offered each quarter during the academic year. It is recommended for students who want to use Python in math, science, or engineering courses and for students who want to learn the basics of Python programming. The goal of the short course is to familiarize students with Pythonâ€™s tools for scientific computing. Lectures will be interactive with a focus on learning by example, and assignments will be application-driven. No prior programming experience is needed. Topics covered include control flow, basic data structures, File I/O, and an introduction to NumPy/SciPy.

Course Policy

There will be seven short homework assignments. You pass the class if you pass each homework assignment which will roughly corresponds to earning at least 70% of the points on each. Homework is assigned for each lecture, and each homework is due by the beginning of the following lecture.

Homework will have starter code / templates (see Assignments) and will be automatically graded at the time of submission. The autograder is there so you can tell if your work is correct and if you have passed the assignment. You can submit any number of times before the due date and you are allowed up to 5 late days. Your final submission will be the one used for a grade.

Gaming the autograder by hard-coding the answers is a violation of the Stanford Honor Code as is using packages which have already implemented what is asked for in a question. You are encouraged to collaborate with others on the homework assignments. However, you must write up your own solutions. You may not copy each others' code.

Assignment Submission:
  1. scp homework file(s) to corn
  2. log into corn and run the following script /afs/ir.stanford.edu/users/d/a/danfrank/cme193/submit.py (easiest if you're in the directory containing the homework files)
  3. follow the prompts
  4. resubmit if you did not pass or want to try for a better score

Development Environment

We will be using NumPy, SciPy, matplotlib, and IPython. in this course. These are all available on corn but often it will be more convenient for you to have them set up on your personal machine. There are numerous tutorials online to get these installed on your system (also see reading for Lecture 1).

In case you are unable to get setup we have created a virtual machine (VM) to work with VirtualBox that has all of the tools needed for the class. The VM uses Linux Debian 6.0.7, the username is cme193 and the password is 193cme. The root password is toor. To get started:

  1. Download and unzip the VM appliance: cme193-debian-6.0.7.ovf.zip (1.9 GB).
  2. Download VirtualBox.
  3. Start VirtualBox and select File --> Import Appliance
  4. Select cme193-debian-6.0.7.ovf for the appliance.
